# Redeeming Telegram

Guide Telegram usage toward accountability, community, and mission alignment.

## Problems

- Scams, noise, and engagement farming
- Fragmented channels and unclear rules

## Redemptive Practices

- Clear code-of-conduct pinned in every channel
- Verified mod roster + escalation routes
- Read-only channels for announcements; thread for Q&A
- Weekly digest summarizing outcomes and proposals
- Bridge key announcements to Discord and docs

## Tech Tools

- Bot to sync proposals from Snapshot
- Link unfurling to docs and dashboards
- Anti-spam, link filters, and onboarding checks

> Technology becomes sacred when it serves clarity and care.
